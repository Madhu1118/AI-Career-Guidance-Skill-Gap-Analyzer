# AI-Based Career Guidance & Skill Gap Analyzer

This project provides an AI-powered solution to help campus students
identify suitable career paths and analyze skill gaps.

## Problem
Students often face confusion in choosing the right career due to lack
of personalized guidance and unclear skill requirements.

## Solution
The system uses AI to analyze student skills, interests, and academic
data to recommend career paths and learning roadmaps.

## Google Technologies Used
- Google Cloud Platform
- Gemini API
- Dialogflow
- Firebase
- Google Colab

## Features
- Personalized career recommendations
- Skill gap analysis
- Learning roadmap suggestions
- AI chatbot support

This repository contains project documentation and MVP design.
